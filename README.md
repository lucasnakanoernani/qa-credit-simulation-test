# Teste Técnico de QA – Simulação de Crédito

## Objetivo
Executar e documentar testes em uma jornada de simulação de crédito, incluindo evidências da execução, criação de casos de teste, registro de bugs (quando aplicável) e desenho do processo no padrão BPMN, incluindo os fluxos As Is (estado atual) e To Be (processo proposto).

## Escopo
- Tipo de crédito: Financiamento Imobiliário
- Canal: Web
- Banco avaliado: Banco do Brasil
- Tipo de teste: Testes funcionais manuais

## Fora de Escopo
- Autenticação e login de usuários
- Validações internas de backend e regras de cálculo não visíveis ao usuário
- Integrações com sistemas internos do banco
- Análise de crédito real e aprovação final

## Ferramentas Utilizadas
- Navegador Web (Google Chrome)
- Documentação em Markdown
- Ferramenta de modelagem BPMN (Bizagi Modeler)

## Premissas
- Os testes foram realizados considerando um usuário não autenticado.
- As informações utilizadas na simulação são fictícias e utilizadas apenas para fins de teste.
- O comportamento do sistema pode variar de acordo com regras internas do banco.

## Sistema Testado
- Nome: Simulação de Financiamento Imobiliário – Banco do Brasil
- URL: https://www.bb.com.br/site/pra-voce/financiamentos/financiamento-imobiliario/
